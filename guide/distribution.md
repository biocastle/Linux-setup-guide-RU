# Выбор дистрибутива Linux
Мои рекомендации в дистрибутивах Linux:
1. [ Arch Linux ](https://archlinux.org) - популярный дистрибутив, который имеет простой и быстрый пакетный менеджер, огромный репозиторий, AUR (Arch User Repository), огромное коммьюнити и вики, который можно использовать для других дистрибутивов.
Если вы не хотите самому настраивать систему с самого начала, то посоветую вам дистрибутив который основан на арче - CachyOS. В нем мы имеем оптимизированное ядро на планировщике BORE, оптимизированные репозиториии и куча другой полезной оптимизации.
2. [ Linux mint ](https://linuxmint.com/) - простой для новичков, стабильный, не особо требующий работы с терминалом дистрибутив где из коробки вам предоставляется окружение рабочего стола Cinnamon.
3. [ Fedora Linux ](https://fedoraproject.org/) - bleeding-edge дистрибутив, который идеально подходит для новичков. Здесь нам предоставляют новые технологии, легкое использование, стабильность.
4. [ NixOS ](https://nixos.org/) - мой любимый дистрибутив, он многим отличается от всех перечисленных. У этого дистрибутива используется декларативный способ настройки системы т.е конфигурация системы в одном файле. Также тут есть свой язык Nix для настройки системы, сборки недостающихся вам пакетов и т.д.
